# Employee Work Tracking System

This project involves designing and implementing a ServiceNow-based Employee Work Tracking System, a custom scoped application designed to streamline employee time management, task assignment, and performance monitoring. The system aims to provide a centralized platform for employees to log their work hours, submit timesheets, and track assigned tasks. Managers will be empowered to approve timesheets, monitor team productivity, and manage employee leave requests. Administrators will have the capability to manage user accounts, generate comprehensive performance reports, and maintain system security. The system will leverage ServiceNow's platform features, including custom tables, workflows, business rules, and access control lists (ACLs) to ensure data integrity, automation, and security. The project will utilize ServiceNow's Flow Designer for automated notifications and potentially incorporate Service Portal for a user-friendly interface.
